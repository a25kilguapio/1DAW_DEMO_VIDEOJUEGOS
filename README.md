# 🧩 CRUD PHP amb MySQLi

## 📌 Descripció

Aquest projecte consisteix en la implementació d’una aplicació **CRUD completa en PHP procedural** utilitzant **MySQLi**, desplegada en un entorn real (`dam.inspedralbes.cat`) i versionada amb **GitHub**.

L’objectiu és entendre el funcionament real d’una aplicació PHP en producció, incloent:

- Connexió a base de dades
- Gestió de formularis
- Execució de consultes SQL
- Desplegament en servidor

---

## 🎯 Objectius

- Desenvolupar una aplicació CRUD funcional
- Aplicar bones pràctiques de codi (**PSR-2**)
- Treballar amb connexions reals a bases de dades
- Utilitzar GitHub amb commits progressius

---

## 🛠️ Tecnologies utilitzades

- **PHP (procedural)**
- **MySQLi**
- **MySQL**
- **HTML / CSS / JS**
- **Git & GitHub**

---

## ⚙️ Funcionalitats (CRUD)

L’aplicació permet:

- ➕ **Create** – Inserir nous registres
- 📄 **Read** – Llistar registres en format taula
- ✏️ **Update** – Editar registres existents
- ❌ **Delete** – Eliminar registres
